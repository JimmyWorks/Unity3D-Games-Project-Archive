Tanks Minigame
Author: Jimmy Nguyen
Email: Jimmy@Jimmyworks.net

How to Play:
Run several instances of Tanks.exe located in Executable folder.
One instance will run the game as the LAN Host(H) server locally.
Other instances must connect to the host via LAN Client(C).

Controls:
Aim using the mouse.
Fire using Spacebar.
Movement with WASD.

Upon death, player will be prompted to respawn.
